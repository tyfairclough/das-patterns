Search

For a simple search of organisations we use this basic pattern.

No results

The layout has 2 key views. The normal results and the last page which has a hook to a manual entry. We've used this pattern to allow as many people through the system as possible.

Number of results

As a rule of thumb we divide 200 by the number of data points in a row to calculate our number of results to display, rounded to the nearest 5 and a maximum of 25 for selections and 50 for data outputs.
