Filters

Filters help users cut large data sets into useful subsets that answer their user need.

Lets talk about the filters generally here.

Filter container

Filters live inside collapsible containers. They're open by default and have a maximum height of 500px. We compartmentalise the filters largely due to their inherit variability. By fixing a max height to the contained content we can ensure other filters are reasonably accesible to the user. The javascript enhancement that allows the filter containers to collapse is a small enhancement that allows users to change the heirarchy of the filters to promote/demote filters that they want.
