Tabs

The Navigation tabs pattern is an exmple of skeuomorphism in the GUI intended to organise related content.

Tabs provide an easy way to show large amounts of similar structured data parted by categories

Tabs create a context for content, when a tab is selected the relevant content is loaded inside the content area.

JS 
	    	
    //tabs pattern
    $(".tabs-menu a").click(function(e) {
        e.preventDefault();
        $(this).parent().addClass("current");
        $(this).parent().siblings().removeClass("current");
        var tab = $(this).attr("href");
        $(".tab-content").not(tab).css("display", "none");
        $(tab).fadeIn();
    });
