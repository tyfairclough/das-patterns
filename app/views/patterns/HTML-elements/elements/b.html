<b>

gov.uk have removed <b> styling through the reset css but have included an override via the use of the .bold-{size} class.

You can also use the decoration class .bold to make any text bold agnostic of its size.

HTML 
                
<p class="bold-xsmall">bold extra small</p>
<p class="bold-small">bold small</p>
<p class="bold-medium">bold medium</p>
<p class="bold-large">bold large</p>
<p class="bold-xlarge">bold extra large</p>
<p class="bold-xxlarge">bold extra extra large</p>
SASS 
                
    // Bold font sizes
.bold-xxlarge {
  @include bold-80();
}

.bold-xlarge {
  @include bold-48();
}

.bold-large {
  @include bold-36();
}

.bold-medium {
  @include bold-24();
}

.bold-small {
  @include bold-19();
}

.bold-xsmall {
  @include bold-16();
}

// Bold, without needing a font size
.bold {
  font-weight: 700;
}
