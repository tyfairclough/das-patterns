Design stage bannersRequires content

We use banners to denote when the service is not live. There are different banners depending on the stage of delivery the project is in.

Prototype banner

When a project is live and you are iterating on the design add the class prototype to the div with the class phase-banner.

PROTOTYPE This is not a real service. Give feedback and help us to build a better service.

HTML 

<div class="phase-banner prototype">
    <p>
        <strong class="phase-tag">PROTOTYPE</strong>
        <span>This is not a real service. <a href="https://www.surveymonkey.co.uk/r/accessmylevysurvey?utm_source=proto&utm_medium=research&utm_campaign=phase-banner" target="_blank">Give feedback</a> and help us to build a better service. </span>
    </p>
</div>
SASS 
        
.phase-banner {
        &.alpha {
            .phase-tag {
            background: $beta-colour
            }
        }
        &.beta {
            .phase-tag {
            background-color: $govuk-blue;
            }
        } 
        &.prototype, &.discovery {
            .phase-tag {
                background-color: $fuschia;
            }            
        }      
    }

Discovery banner

When in discovery add discovery to the div with the class phase-banner.

DISCOVERY This is not a real service. Give feedback and help us to build a better service.

Alpha banner

When in alpha add alpha to the div with the class phase-banner.

ALPHA This is a new service - your feedback will help us improve it.

Beta banner

When in beta add beta to the div with the class phase-banner.

BETA This is a new service - your feedback will help us improve it.
