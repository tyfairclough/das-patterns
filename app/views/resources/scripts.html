Run scripts on the right page

This script reads the <main> element's class name and looks for it in a switch statement. If a match is found it executes a function within which your pages javascript can reside.

            
switch (className) {
   case 'a-page':
       aPage();
   break;
   case 'finance-page':
       financePage();
   break;
   default: break;
};

function financePage(){};

Redirect users when they use a radio select list

This script reads the <main> element's class name and looks for it in a switch statement. If a match is found it executes a function within which your pages javascript can reside.

            
 $(".button").click(function(e){
    e.preventDefault(); 
    state = $("input[name=radio-group]:checked").val()                  
    switch (state) {
        case 'yes':
            // do something if they select the "yes" options
            window.location.href = "go-to-a-page";             
        break;
        case 'no':
            // do something if they select the "no" options
            window.location.href = "go-to-a-page";   
        break;
        default: 
            // if they select nothing 
            alert("select an option");
        break;
    }                        
});
